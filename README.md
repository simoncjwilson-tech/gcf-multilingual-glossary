# GCF Multilingual Glossary

A lookup tool for Green Climate Fund acronyms and key terms across the five other UN languages
(French, Spanish, Russian, Arabic, Chinese).

**Live tool:** open `index.html` (self-contained — no server needed) or visit the GitHub Pages link.

- 88 acronyms + 53 key terms, each tagged with its source document.
- Data: `glossary-data.json`. Sources: GCF Programming Manual, Investment Framework for GCF-2, Accreditation Framework of the GCF.

Prototype — not an official GCF product.
