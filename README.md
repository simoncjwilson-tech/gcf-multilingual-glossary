# GCF Multilingual Glossary

A lookup tool for Green Climate Fund acronyms and key terms across the five other UN languages
(French, Spanish, Russian, Arabic, Chinese).

**Live tool:** <https://simoncjwilson-tech.github.io/gcf-multilingual-glossary/>
(or open `index.html` locally — self-contained, no server needed).

- **374 entries** — 121 acronyms + 253 key terms — each tagged with its source document.
- **6 languages:** English, French, Spanish, Russian, Arabic, Chinese (EN/FR/ES/RU/AR/ZH).
- Compiled from **52 official GCF documents** published on greenclimate.fund.
- **Human-translated:** every translation is taken from official GCF documents, professionally
  translated by a translation agency and proof-read by a second agency. No machine or AI
  translation is used.
- Data: `glossary-data.json`.

Prototype — not an official GCF product.
